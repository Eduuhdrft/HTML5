Estilizando imagens

Usando a tag img

Object-fit: cover

A imagem preenche o container, mantendo a porpoção e cortandose necessário

Objetc fit: contain

A imagem é redimensionada para caber inteiramente no container, mantendo a proporção

Objetc fit: fill

A imagem é esticada para preencher o container, podendo distorcer a proporção

Usando background-image

background-sinze: cover

A imagem de fundo cobre todo o container, mantendo a proporção e cortando se necessário

background--size: contein

A imagem é redimensionada para caber inteiramente no container, mantendo a proporção
